## Description

Projeto desenvolvido durante o Ignite Lab - NodeJs da [Rocktseat](https://app.rocketseat.com.br/).

## Installation

Clone o projeto

```bash
  git clone https://github.com/valdir-alves3000/notifications-service.git
```

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
